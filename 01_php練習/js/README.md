# VS Code → エクスプローラー ファイルコピー ツール

VS Code で開いているファイルを **Windowsのクリップボード**に乗せて、
エクスプローラーで `Ctrl + V` で貼れるようにするツールです。

「VS Codeでコピー → 物理フォルダで貼り付け」を1つのホットキーで実現します。

---

## 🎯 使い方（3ステップ）

```
① VS Code で対象ファイルを開く（タブをアクティブに）
   ↓
② Ctrl + Alt + C を押す   ← 「コピー完了」と一瞬出る
   ↓
③ エクスプローラーの入れたいフォルダで Ctrl + V
```

- **フォルダのパスでもOK**（そのフォルダごとコピーされます）
- 貼り付け先は普通の右クリック→貼り付けでもOK

---

## ⚙️ 仕組み（なぜ動くか）

VS Code の `Ctrl+C` は「VS Code専用の箱」に入るため、エクスプローラーでは貼れません。
このツールは箱をつなぎます👇

```
Ctrl+Alt+C を押す
  ↓ VS Code に「今開いてるファイルのパス」をコピーさせる
  ↓ そのファイルを Windows のクリップボードに乗せ直す（PowerShell の Set-Clipboard）
  ↓
エクスプローラーで Ctrl+V → ちゃんと貼れる
```

---

## 🚀 起動方法

### 今すぐ使う
`vscode-to-explorer.ahk` を **ダブルクリック**するだけ（AutoHotkey v2 が必要）。
タスクトレイに緑のHアイコンが出れば起動中です。

### パソコン起動時に自動で立ち上げる
1. `Win + R` → `shell:startup` と入力 → Enter（スタートアップフォルダが開く）
2. `vscode-to-explorer.ahk` の **ショートカット**をそのフォルダに入れる

これで再起動しても自動で有効になります。

---

## 🔧 カスタマイズ

`vscode-to-explorer.ahk` の中の記号を変えるとホットキーを変更できます。

| 記号 | キー |
|------|------|
| `^` | Ctrl |
| `!` | Alt |
| `+` | Shift |
| `#` | Windows |

例：`^!c::` を `^!v::` にすると `Ctrl + Alt + V` に変わります。

---

## ⚠️ 注意点・つまずきポイント

- **ファイルを開いた状態**（エディタでアクティブ）で押すこと。サイドバーで選んだだけだと取れない場合があります。
- VS Code の「Copy Path of Active File」= `Ctrl+K Ctrl+Shift+C` を使っています。ここのキー割り当てを変えている場合は動きません。
- 他のAHKスクリプトで `Ctrl+Alt+C` を既に使っていると衝突します（その場合は上のカスタマイズでキー変更）。
- 反応しないときは、タスクトレイのアイコンを右クリック → Reload でスクリプトを再読み込み。
